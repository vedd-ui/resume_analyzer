# resume_analyzer
NLP Based Resume Analyzer
Resume Analyzer (NLP Project)

Overview: This project is an NLP-based Resume Analyzer that evaluates how well a resume matches a given job description. It mimics basic ATS (Applicant Tracking System) functionality.

Features
- Extracts skills from resume text
- Supports skill variations (e.g., ML for Machine Learning)
- Calculates skill frequency
- Matches resume with job description
- Computes match percentage
- Provides feedback (Excellent / Moderate / Low match)

Approach
1. Preprocess text (lowercase, remove noise)
2. Extract skills using keyword matching
3. Handle variations of skills
4. Compare resume with job description
5. Calculate match percentage
